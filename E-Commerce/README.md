## 📊 Hướng dẫn chuẩn bị dữ liệu (Data Setup)

Vì tập dữ liệu lịch sử giao dịch và biến động logistics thương mại điện tử thời trang có dung lượng lớn để mô hình Prophet + LightGBM dự báo doanh thu hằng ngày, toàn bộ các tệp dữ liệu gốc `.csv` đã được lưu trữ an toàn trên Google Drive nhằm tối ưu hóa bộ nhớ cho mã nguồn sạch.

Để phân tích và thực thi mô hình, bạn vui lòng:
1. Tải toàn bộ các file dữ liệu gốc tại đây: https://drive.google.com/drive/folders/1Jh4ednIPqNSg41bHIzh4_1GnG9yMNfSH?usp=drive_link
2. Tạo một thư mục con tên là `data/` nằm ngay bên trong thư mục chứa mã nguồn trên máy tính cá nhân của bạn.
3. Di chuyển hoặc giải nén tất cả các file dữ liệu vừa tải vào thư mục `data/` đó trước khi khởi chạy các file Jupyter Notebook (`.ipynb`).
